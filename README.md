# Nasa Asteroides
SpringBoot Api

## Requirements

1. Java 8+
2. Maven

## How to run

1. Clone

```
   git clone https://github.com/carolinedesousasantos/nasa-asteroides.git
```

3. Go into folder

```
   cd nasa-asteroides
```

4. Execute

```
   mvn spring-boot:run
 ```

3. Open the link on browser

   [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
